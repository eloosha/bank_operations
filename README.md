# Учебный проект по Python "bank_operations".

## Описание:

bank_operations - это виджет, который показывает несколько успешных банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:

```
git clone https://github.com/eloosha/bank_operations.git
```

2. Установите зависимости:

```
pip install -r requirements.txt
```

3. Создайте базу данных и выполните миграции:

```
python manage.py migrate
```

4. Запустите локальный сервер:

```
python manage.py runserver
```

## Использование:

1. В модуле masks.py находятся функции маскировки номеров карты или счета.
2. В модуле widget.py находится функция которая принимает тип карты/счета, номер карты/счета и маскирует их с помощью
   функций из masks.py
3. В модуле processing.py находятся функции которые фильтруют данные по обработанным и по дате.
Так же функция которая фильтрует список по ключевому слову в описании операции и
функция 
4. В модуле generators.py находятся функции которые выдают транзакции по определенной валюте,
выдает описание транзакций и функция которая генерирует номера банковских карт.
5. В модуле decorators.py находится декоратор для функции, который логирует действия функции.
6. В модуле utils.py находятся функции определения формата файла и считывание его данных для использования в дальнейших функциях.
7. В модуле main.py находится основная логика программы и её запуск.
